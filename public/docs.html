Images Processing

This website compress and process your image and convert their format live

  1. https://images-process.herokuapp.com/url?imageUrl=YourImage_URL
    2. Options in query:
    imageWidth // number of pixels
    imageHeight // number of pixels
    imageFormat // png, webp, jpg, jpeg
    response will be an image you can use it in image src directly
  1. https://images-process.herokuapp.com/data-base64?imageUrl=YourImage_URL
    2. Options in query:
    imageWidth // number of pixels
    imageHeight // number of pixels
    imageFormat // png, webp, jpg, jpeg
    response will be an base64 data you can use it in image src directly {image: 'data'}

Examples:

You can use the second link only direct in browser, and you can use http request for both ways by Method: Get or in src attribute og image tag
